# StackField

A free, ad-free puzzle game where you place blocks on an 8x8 grid to clear rows and columns.

## How to Play

1. Drag blocks from the bottom area onto the grid
2. Fill complete rows or columns to clear them and score points
3. Clear multiple lines at once for combo bonuses
4. Game ends when no more blocks can be placed

## Installation

### Play in Browser
Simply open `index.html` in any modern browser, or deploy to GitHub Pages.

### Install on iOS (iPhone/iPad)
1. Open the game in Safari
2. Tap the Share button (square with arrow pointing up)
3. Scroll down and tap "Add to Home Screen"
4. Tap "Add" to confirm
5. The app icon will appear on your home screen

### Install on Android
1. Open the game in Chrome
2. Tap the three-dot menu in the top right
3. Tap "Add to Home screen" or "Install app"
4. Confirm the installation
5. The app icon will appear on your home screen

## Deploy to GitHub Pages

1. Push this repository to GitHub
2. Go to repository Settings > Pages
3. Under "Source", select "Deploy from a branch"
4. Select "main" branch and "/ (root)" folder
5. Click Save
6. Your game will be available at `https://yourusername.github.io/StackField/`

## Features

- No ads
- Works offline (PWA)
- Installable on iOS and Android
- Touch and mouse support
- High score tracking
- Combo system for clearing multiple lines

## Tech Stack

- Vanilla JavaScript
- HTML5 / CSS3
- Progressive Web App (PWA)
- Service Worker for offline support
